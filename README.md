## 📌 Overview

A modern, responsive, and fully functional **personal portfolio website** built to showcase my skills, projects, experience, and achievements as a **Computer Engineer & Full Stack Developer**.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🏠 Hero Section | Full-screen background image with smooth scroll CTA buttons |
| 👤 About Me | Profile photo, biography, CV download, skills overview & certificates |
| 🗂️ Projects | 14 real GitHub projects with filters, pagination, GitHub & Live Demo links |
| 🛠️ Skills | Progress bars, skill tags, grouped tabs (Frontend, Backend, Databases, Tools, Languages) |
| 💼 Experience | 5 professional roles with dot pagination |
| 📝 Blog | 5 technical posts with category filter, full post modal, code snippets & comments |
| 🌟 Testimonials | 6 real endorsements with dot pagination |
| 🤖 AI Feature | Personalized skill & project recommendations based on user interests |
| 📬 Contact | EmailJS-powered form that sends messages directly to inbox |
| 🌙 Dark/Light Mode | Toggle between dark and light themes |
| 📱 Responsive | Mobile-first design for all screen sizes |
